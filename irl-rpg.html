IRL.RPG

App Design

Project Description

Home page

  This project was an app (re)design for a mobile application of our choosing for IDM 213 (Interaction Design). We were tasked with redesigning, rebranding, and ultimately reimagining an app, using Figma to develop our vision for the core task a user could perform on the new app. After fleshing out interactions and microinteractions with Figma, the final task flow would then be animated using Adobe After Effects.

Creating a task

  The app I created is IRL.RPG, a habit-building and productivity app inspired by Habitica that gamifies users' everyday tasks through a range of retro RPG elements. With a user-created avatar, users complete tasks and chores for in-app currency, items, and experience for a true RPG experience. IRL.PRG uses a blend of productivity features, game-like features, and a pixel art style to create a novel task-managing experience.
